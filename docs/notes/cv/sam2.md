# Segment Anything Model 2 (SAM 2)

> 论文：Ravi et al., 2024, "SAM 2: Segment Anything in Images and Videos", arXiv:2408.00714, Meta AI

## 1. 研究背景与动机

视频理解（Video Understanding）是计算机视觉领域最具挑战性的任务之一，其核心难点在于如何在时间维度上保持分割的一致性和连续性。传统的视频目标分割（Video Object Segmentation, VOS）方法通常针对特定类别的物体进行优化，难以泛化到新类别。

**Segment Anything Model (SAM)** 在图像分割领域取得了突破性进展，但其设计初衷主要面向静态图像，无法有效处理视频中的时序信息。SAM 2的核心动机是将SAM的"Segment Anything"范式扩展到视频领域，实现**图像和视频的统一分割**。

SAM 2面临的核心挑战包括：

1. **时序一致性（Temporal Consistency）**：同一物体在不同帧中应保持一致的分割结果
2. **遮挡处理（Occlusion Handling）**：物体被遮挡后重新出现时需要正确重识别
3. **运动模糊（Motion Blur）**：快速运动导致的图像模糊增加了分割难度
4. **计算效率（Computational Efficiency）**：视频处理需要考虑实时性要求

## 2. 核心贡献

SAM 2的核心贡献可以概括为三个方面：

1. **统一架构**：设计了一种同时处理图像和视频分割的统一架构
2. **记忆机制（Memory Mechanism）**：提出基于记忆模块的视频分割范式
3. **大规模数据**：发布了包含稀疏标注的SA-V数据集

## 3. 技术架构详解

### 3.1 整体架构

SAM 2采用与SAM相似的encoder-decoder架构，并额外引入了视频处理所需的记忆模块：

```
图像编码器 → 提示编码器 → 记忆编码器 → 掩码解码器
     ↓
  记忆银行（Memory Bank）
```

#### 3.1.1 图像编码器（Image Encoder）

SAM 2的图像编码器与SAM相同，采用MAE预训练的ViT-H/ViT-L模型。图像编码器的输出是帧的视觉特征表示，用于后续的提示编码和掩码解码。

#### 3.1.2 提示编码器（Prompt Encoder）

SAM 2的提示编码器支持两种类型的提示：

- **空间提示（Spatial Prompt）**：与SAM相同的点、框、涂鸦提示
- **时间提示（Temporal Prompt）**：SAM 2的新增特性，允许在任意帧指定目标，然后传播到整个视频序列

时间提示的概念是SAM 2的核心创新之一。用户可以在任意帧点击目标物体，模型会自动将分割结果传播到视频的其他帧。

#### 3.1.3 记忆编码器（Memory Encoder）

记忆编码器是SAM 2的核心创新，用于将当前帧的预测结果编码为可供后续帧使用的记忆表示。

当处理完第 $t$ 帧后，记忆编码器将：

1. 当前帧的分割掩码 $M_t$
2. 当前帧的图像特征 $F_t$
3. 当前帧的IoU预测分数

编码为记忆向量 $h_t \in \mathbb{R}^{D_h}$，存储到记忆银行（Memory Bank）中。

### 3.2 记忆机制（Memory Mechanism）

记忆机制是SAM 2处理视频分割的核心技术，其设计借鉴了人体视觉系统的工作记忆（Working Memory）概念。

#### 3.2.1 记忆银行（Memory Bank）

记忆银行存储了历史帧的分割信息和记忆向量。对于长度为 $T$ 的视频，已处理帧的记忆表示为：

$$\mathcal{M} = \{h_1, h_2, ..., h_{t-1}\}$$

记忆银行采用FIFO（First-In-First-Out）策略，当存储空间不足时，丢弃最旧的记忆。

#### 3.2.2 记忆注意力（Memory Attention）

在处理新帧时，SAM 2通过**记忆注意力**机制检索相关记忆：

1. 将当前帧的图像特征作为Query
2. 将记忆银行中的记忆作为Key和Value
3. 通过cross-attention机制融合历史信息

记忆注意力的计算过程为：

$$\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

其中 $Q$ 来自当前帧，$K$ 和 $V$ 来自记忆银行。这种机制使模型能够：

- 利用前期帧的分割结果指导当前帧预测
- 处理物体遮挡和重新出现
- 保持跨帧的实例一致性

#### 3.2.3 记忆选择策略

并非所有历史帧都具有同等重要性。SAM 2采用了**稀疏记忆选择**策略：

- **关键帧（Key Frame）**：选择包含目标物体清晰外观的帧
- **最近帧（Recent Frame）**：优先保留最近的分割结果
- **高置信度帧**：保留模型高置信度的预测结果

### 3.3 掩码解码器（Mask Decoder）

SAM 2的掩码解码器同样采用Transformer decoder架构，但针对视频任务进行了调整：

1. **多帧输入**：解码器不仅接收当前帧的图像嵌入和提示嵌入，还接收记忆注意力模块的输出
2. **实例级别的输出**：对于多目标场景，解码器输出实例ID，实现跨帧的目标追踪
3. **置信度预测**：输出当前预测的IoU置信度，用于指导记忆选择

### 3.4 流式推理（Streaming Inference）

SAM 2支持两种推理模式：

#### 3.4.1 离线模式（Offline Mode）

已知完整视频，提前处理所有帧。离线模式可以进行全局优化，如双向传播和平滑处理。

#### 3.4.2 在线模式（Online Mode）

实时处理视频流，仅利用历史帧信息。在线模式对实时应用至关重要，如视频编辑和自动驾驶。

## 4. 训练策略

### 4.1 视频-图像联合训练

SAM 2采用图像和视频联合训练的策略：

- **图像数据**：使用SA-1B数据集，与SAM相同
- **视频数据**：使用新收集的SA-V数据集

联合训练确保模型同时具备图像分割和视频分割能力。

### 4.2 记忆规模控制

训练时对记忆银行的大小进行控制：

- 图像编码器每帧运行一次
- 记忆编码器仅处理最近 $N$ 帧（通常 $N=6$）
- 记忆银行的容量限制模拟了实际部署场景

### 4.3 数据增强

视频分割训练采用以下数据增强策略：

- **时间步采样**：随机跳帧训练模型处理长距离依赖
- **遮挡模拟**：随机遮盖目标物体，训练模型重识别能力
- **运动模糊模拟**：对图像施加模糊，训练鲁棒性

## 5. 数据集：SA-V

SAM 2发布了**SA-V数据集**，这是目前规模最大的视频分割标注数据集：

- 约51,000个视频，涵盖多样化的场景
- 超过200万个掩码标注
- 视频长度从几秒到几分钟不等
- 包含丰富的物体类别和交互场景

SA-V采用稀疏标注策略：并非所有帧都标注，仅标注关键帧，然后通过插值获得连续掩码。

## 6. 实验结果与分析

### 6.1 主要基准测试

SAM 2在多个视频分割基准上取得了SOTA（State-of-the-Art）性能：

#### 6.1.1 DAVIS Video Object Segmentation

DAVIS是视频目标分割的标准基准。SAM 2在DAVIS 2017上实现了显著的性能提升，平均Jaccard指数（$\mathcal{J}$）和轮廓精度（$\mathcal{F}$）均超过先前方法。

#### 6.1.2 SAM-VOS（Video Object Segmentation）

SAM-VOS是针对SAM能力评估设计的基准，包含未见过的物体类别。SAM 2展现了强大的泛化能力，在类别无关设置下表现优异。

#### 6.1.3 UVO（Unidentified Video Objects）

UVO是开放世界视频分割基准，SAM 2在各类别上均展现了竞争力的性能。

### 6.2 消融实验（Ablation Study）

论文进行了详尽的消融实验，验证各组件的贡献：

- **记忆机制**：移除记忆机制后，性能显著下降，证明时序信息的重要性
- **记忆规模**：更大的记忆容量带来边际收益递减
- **图像编码器规模**：ViT-H优于ViT-L，但计算成本更高

### 6.3 定性分析

SAM 2在以下困难场景表现突出：

- **长时间遮挡**：目标被完全遮挡后重新出现时，能正确追踪
- **外观变化**：目标物体发生剧烈外观变化（如姿态、尺度）时保持追踪
- **多目标交互**：多个相似物体相互遮挡时，保持实例一致性

## 7. 与SAM的对比分析

| 特性 | SAM | SAM 2 |
|------|-----|-------|
| 任务范围 | 图像分割 | 图像+视频统一分割 |
| 架构组成 | 图像编码器+提示编码器+掩码解码器 | 图像编码器+提示编码器+记忆编码器+掩码解码器 |
| 提示类型 | 空间提示（点、框、文本） | 空间提示+时间提示 |
| 输出 | 分割掩码 | 分割掩码+实例ID |
| 时序建模 | 无 | 记忆注意力机制 |
| 视频支持 | 逐帧处理，无记忆 | 流式处理，支持在线/离线 |
| 遮挡处理 | 不适用 | 通过记忆重识别 |

## 8. 应用场景

SAM 2的潜在应用场景包括：

1. **视频编辑**：交互式视频物体分割和替换
2. **自动驾驶**：道路场景的实时分割和追踪
3. **医学影像**：CT/MRI序列中的器官和病变分割
4. **视频标注**：加速大规模视频数据的标注效率
5. **增强现实**：实时视频中的物体追踪和虚拟叠加

## 9. 局限性与未来方向

### 9.1 当前局限性

- **计算资源需求**：大型ViT模型和记忆机制带来较高延迟
- **复杂场景失败**：在极度拥挤或相似物体密集的场景可能失败
- **文本理解能力**：文本提示的语义理解能力仍有提升空间
- **长视频处理**：超长视频的记忆衰减问题

### 9.2 未来研究方向

- 更高效的记忆机制设计
- 联合语音-视觉-文本的多模态分割
- 3D场景理解和分割
- 端到端的视频实例分割和追踪

## 10. 学术影响与意义

SAM 2的学术贡献主要体现在：

1. **统一范式的扩展**：将图像分割的成功范式扩展到视频领域
2. **记忆机制的设计**：为视频理解提供了新的技术路径
3. **开源生态建设**：发布的SA-V数据集和模型权重推动了领域发展
4. **多模态融合**：探索了视觉和语言在视频分割中的结合

SAM 2的出现标志着视频分割进入了一个新的时代，为实现"Segment Anything in Images and Videos"的愿景迈出了重要一步。
