# ResNet: Deep Residual Learning for Image Recognition

> 论文：He et al., 2016, "Deep Residual Learning for Image Recognition", CVPR
> 论文链接：https://arxiv.org/abs/1512.03385

## 1. 研究背景与问题定义

### 1.1 深度学习与卷积神经网络的演进

卷积神经网络（Convolutional Neural Network, CNN）在计算机视觉领域的发展历程中，深度是一个关键因素。从LeNet（5层）到AlexNet（8层），再到VGGNet（16-19层），网络深度的增加带来了性能的显著提升。这一观察促使研究者们相信：**网络越深，其表达能力越强，能够学习的特征层次越丰富**。

在图像分类任务中，网络浅层通常学习低层特征（边缘、纹理），深层学习高层语义特征（物体部件、整体结构）。因此，增加网络深度理论上应该能够帮助网络学习更复杂的特征表示。

### 1.2 退化问题（Degradation Problem）

然而，He等人（2015）发现了一个反直觉的现象：当网络深度超过某个临界点后，增加更多层反而导致训练误差和测试误差增加，这被称为**退化问题（Degradation Problem）**。

退化问题与梯度消失/爆炸（Vanishing/Exploding Gradients）不同：

- **梯度消失/爆炸**：发生在网络极深时，梯度在反向传播过程中指数级衰减或爆炸，导致网络难以训练
- **退化问题**：即使使用Batch Normalization等归一化技术缓解了梯度问题，更深的网络仍然表现更差

这一现象表明，退化问题并非源于优化困难，而是深层网络的**表达能力**或**学习能力**不足。

### 1.3 学术问题形式化

假设浅层网络学习到了最优函数映射 $\mathcal{H}(x)$。对于深层网络，我们期望其深层部分能够学习到恒等映射（Identity Mapping）$\mathcal{H}(x) = x$，使深层网络退化为浅层网络。然而，实验表明随机初始化的深层网络难以学习到这种恒等映射。

## 2. 核心贡献：残差学习框架

### 2.1 残差连接（Residual Connection）的数学形式

ResNet的核心思想是引入**残差连接（Residual Connection）**，也称为**跳跃连接（Skip Connection）**或**捷径连接（Shortcut Connection）**。

传统网络学习底层映射 $\mathcal{H}(x)$。残差网络则让网络学习残差（Residual）$\mathcal{F}(x) = \mathcal{H}(x) - x$，最终输出为：

$$\mathcal{H}(x) = \mathcal{F}(x) + x$$

其中：
- $x$ 是输入（也称为恒等分支或identity分支）
- $\mathcal{F}(x)$ 是学习到的残差映射（通常包含2-3层网络）
- $\mathcal{F}(x) + x$ 通过跳跃连接实现逐元素加法

当残差 $\mathcal{F}(x) = 0$ 时，输出即为恒等映射 $\mathcal{H}(x) = x$。这意味着学习 $\mathcal{F}(x) = 0$ 比学习恒等映射更容易。


在实践中，残差块通常包含两个卷积层，因此 $\mathcal{F}(x)$ 通常表示为：

$$\mathcal{F}(x) = \mathcal{W}_2 \cdot \sigma(\mathcal{W}_1 \cdot x)$$

其中 $\mathcal{W}_1$ 和 $\mathcal{W}_2$ 是可学习的权重，$\sigma$ 是ReLU激活函数。

## 3. 网络架构设计

### 3.1 残差块变体：Basic Block与Bottleneck Block

ResNet提供了两种残差块实现：

#### 3.1.1 Basic Block（基本残差块）

Basic Block用于较浅的网络（ResNet-18/34），包含两个3×3卷积层：

```
输入 x
  ↓
3×3 Conv → BN → ReLU → 3×3 Conv → BN
  ↓                                    ↓
              ↓ ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←↗↘ (Add)
输出 H(x)
```

#### 3.1.2 Bottleneck Block（瓶颈残差块）

Bottleneck Block用于较深的网络（ResNet-50/101/152），采用1×1→3×3→1×1的卷积堆叠：

```
输入 x
  ↓
1×1 Conv (降维) → BN → ReLU
  ↓
3×3 Conv → BN → ReLU
  ↓
1×1 Conv (升维) → BN
  ↓                                    ↓
              ↓ ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←↗↘ (Add)

```

Bottleneck设计的优势：

- 1×1卷积层可以显著减少参数量
- 中间3×3卷积在压缩后的特征空间上进行计算
- 最后的1×1卷积恢复通道维度

## 4. 关键技术分析

### 4.1 逐元素加法（Element-wise Addition）与维度匹配

残差连接中的逐元素加法要求主分支和捷径分支的输出维度匹配。当输入输出维度不同时，ResNet采用两种策略：

1. **零填充捷径（Zero-padding Shortcut）**：使用零填充扩展维度，不引入额外参数
2. **投影捷径（Projection Shortcut）**：使用1×1卷积变换维度，引入额外参数

实验表明，投影捷径在准确率上略优于零填充，但由于参数量增加，实际应用中零填充更为常见。

### 4.2 Batch Normalization的协同作用

ResNet的成功离不开**Batch Normalization (BN)**的协同作用。BN在每个mini-batch上对层的输入进行归一化：

$$\mu_{\mathcal{B}} = \frac{1}{m} \sum_{i=1}^{m} x_i, \quad \sigma_{\mathcal{B}}^2 = \frac{1}{m} \sum_{i=1}^{m}(x_i - \mu_{\mathcal{B}})^2$$

$$\hat{x}_i = \frac{x_i - \mu_{\mathcal{B}}}{\sqrt{\sigma_{\mathcal{B}}^2 + \epsilon}}, \quad y_i = \gamma \hat{x}_i + \beta$$

其中 $\gamma$ 和 $\beta$ 是可学习的缩放和偏移参数。

BN的作用：
- 缓解内部协变量偏移（Internal Covariate Shift）
- 使梯度更加稳定
- 允许使用更高的学习率
-具有一定的正则化效果

### 4.3 梯度流动分析

残差连接的一个重要优势是改善了梯度流动。考虑第 $l$ 层的梯度：

$$\frac{\partial \mathcal{L}}{\partial x_l} = \frac{\partial \mathcal{L}}{\partial x_L} \cdot \frac{\partial x_L}{\partial x_l}$$

其中 $L$ 是网络深度。根据链式法则：

$$\frac{\partial x_L}{\partial x_l} = \prod_{i=l}^{L-1} \frac{\partial x_{i+1}}{\partial x_i} = \prod_{i=l}^{L-1} (\mathcal{F}_i' + 1)$$

由于存在"+1"项，即使 $\mathcal{F}_i'$ 的值很小，梯度也不会消失。这使得深层网络的训练更加稳定。

## 5. 实验结果与分析

### 5.1 ImageNet分类实验

ResNet在ImageNet分类任务上取得了突破性成果：

| 模型 | Top-1 Error | Top-5 Error | 参数量 |
|------|-------------|-------------|--------|
| VGGNet-19 | 28.5% | 9.6% | 138M |
| GoogLeNet | 26.4% | 8.8% | 6.8M |
| ResNet-34 | 25.3% | 8.0% | 21.8M |
| ResNet-50 | 22.5% | 6.5% | 25.6M |
| ResNet-101 | 21.8% | 6.3% | 44.5M |
| ResNet-152 | 21.4% | 5.8% | 60.2M |

值得注意的是，ResNet-152的参数量远少于VGGNet-19，但性能显著更好。

### 5.2 CIFAR-10消融实验

论文在CIFAR-10数据集上进行了详尽的消融实验：

- **残差连接的重要性**：移除残差连接后，110层网络的性能远差于20层网络
- **残差学习的效果**：学习残差比学习原始映射更容易收敛
- **网络深度的影响**：残差网络随深度增加性能持续提升

### 5.3 可视化分析

通过可视化学习到的特征，ResNet展现了与浅层网络不同的特性：
- 浅层残差块学习低层特征（边缘、纹理）
- 深层残差块学习高层特征（物体部件）
- 残差连接使得信息可以直接跨越多层传递

## 6. 变体与后续发展

### 6.1 ResNet变体

| 变体 | 改进点 |
|------|--------|
| Pre-act ResNet | BN和ReLU在卷积之前 |
| ResNeXt | 引入Cardinality维度 |
| Wide ResNet | 增加通道数，减少深度 |
| DenseNet | 密集连接替代残差连接 |
| Res2Net | 多尺度特征融合 |
| RepVGG | 重参数化技术 |

### 6.2 学术影响

ResNet对深度学习领域产生了深远影响：

1. **残差连接成为标准组件**：几乎所有现代CNN架构都采用残差连接
2. **推动极深网络发展**：使得训练数百层的网络成为可能
3. **启发Transformer设计**：残差连接被广泛应用于Transformer架构
4. **开创深度学习新范式**：证明了"更宽/更深"不一定是最好

## 7. 优势与局限性

### 7.1 优势

- **解决退化问题**：首次实现超深网络（1000+层）的有效训练
- **计算效率高**：相比同等深度的plain网络，参数更少
- **泛化能力强**：在多个视觉任务上取得SOTA
- **迁移学习友好**：预训练权重被广泛用于下游任务

### 7.2 局限性

- **参数量仍然较大**：152层网络有60M参数
- **推理速度较慢**：相比轻量级网络不够高效
- **残差连接的开销**：内存访问成本较高
- **通道维度必须匹配**：需要仔细设计维度变换

## 8. 总结

ResNet通过引入残差连接，有效解决了深层网络的退化问题，使得训练数百层的神经网络成为可能。其核心思想——学习残差而非完整映射——不仅在计算机视觉领域产生了深远影响，还被广泛应用于自然语言处理、语音识别等其他领域。残差连接已成为现代深度学习架构的基本组件之一。
