# YOLO 系列

You Only Look Once (YOLO) 是实时目标检测的经典算法。

## YOLOv1 (2016)

- 首次实现端到端的目标检测
- 将检测任务作为回归问题处理
- 速度快但精度较低

## YOLOv3 (2018)

- 引入多尺度预测（FPN）
- Darknet-53 backbone
- 平衡了精度和速度

## YOLOv5 (2020)

- PyTorch 实现
- 丰富的模型尺寸（n/s/m/l/x）
- 工程化程度高，应用广泛

## YOLOv8 (2023)

- 最新的 Anchor-Free 版本
- 更高的精度和速度
- 统一了分类和检测任务

## 核心思想

将图像划分为 S×S 网格，每个网格预测 B 个边界框及其置信度和类别概率。
